# Netflix Data Analysis Project

## Overview
This project analyzes Netflix content data using Python, MySQL, and Power BI.

## Technologies Used
- Python
- Pandas
- NumPy
- MySQL
- Power BI
- OpenPyXL

## Project Workflow

### Step 1: Data Cleaning & Feature Engineering
- Missing value handling
- Date parsing
- Audience group creation
- Content age calculation
- Genre and country expansion

### Step 2: MySQL Integration
- Database schema creation
- Data loading into MySQL
- SQL views creation

### Step 3: Power BI Dashboard
- KPI Cards
- Genre Analysis
- Country Analysis
- Content Growth Trends
